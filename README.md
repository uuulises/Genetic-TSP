# Genetic-TSP

todos los parametros son configurables, los generales (tamaño poblacion y condicion de corte (max generaciones)) a través d econstantes y los específicos de los componentes, al momento de crearlos.

ejecutar una configuración específica (modificando Main): mvn clean compile exec:java -Pmain
ejectuar experimentos: mvn clean compile exec:java -Pexperiments
